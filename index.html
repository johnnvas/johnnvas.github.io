Intro

I'm Johnn Vasquez and I create web applications for any purpose. I am Colombian-American currently in Boston, MA but work with people all over the country.

My interests include technology, web development, and dentistry. As a former aspiring dentist and oral surgery assistant, I have always searched to be challenged mentally and academically. I was constantly researching new trends and up-and-coming procedures in the dental world to see how I can help improve the productivity of the clinic I was working with at the time. It was around that time that my love for programming began. I was amazed with the role developers played in technological advancements around the world. I found it fascinating how front-end devs could create a beautiful, interactive website, or how Python devs can create autonimous programs with machine learning. It was then that I decided to become a web developer.

Work

Some of my finished applications include Med.io, a React/Python web application that utilizes a medical API to help facilitate communication between patients and doctors. GrammyGram, a React instagram clone dedicated to the Grammy's, and GoodClips, a GoodReads clone that allows users to rate and review some of their favorite movies.

Skills

When my web development journey began with HTML and CSS I quickly learned that was just the literal tip of the iceberg. After tackling Javascript I began looking for new challenges and new languages that can further propel me into the working developer world. Download my resume!
